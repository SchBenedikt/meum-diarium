# **App Name**: Meum Diarium

## Core Features:

- Author Selection: Select an author (e.g., Caesar, Cicero) to view their diary entries and scientific articles.
- Diary Entry Creation: Create diary entries from the perspective of the selected author.
- Scientific Article Creation: Create scientific articles with a neutral, fact-based perspective.
- Content Filtering: Filter content by type: diary entries or scientific articles.
- Dynamic Theme Adjustment: Automatically adjust the website's design based on the selected author, creating a unique visual experience for each persona. An AI tool will apply style choices that harmonize with each historical persona.

## Style Guidelines:

- Primary color: Deep, regal purple (#6A4C93), reminiscent of Roman emperors, for a sense of history and importance.
- Background color: Soft lavender (#E6E6FA), a desaturated, lighter tint of the primary, providing a calming backdrop.
- Accent color: Golden yellow (#FFC107), inspired by ancient Roman ornaments, to highlight important elements and call to actions.
- Body and headline font: 'Literata', a serif font for a literary, vintage feel that evokes the ancient world.
- Use classical Roman-style icons (e.g., laurel wreaths, Roman numerals) to enhance the historical theme.
- Employ a clean, modern layout that prioritizes readability and intuitive navigation. Use of columns and sections to separate diary entries from scientific content.
- Subtle page transitions and hover effects to enhance user engagement.
