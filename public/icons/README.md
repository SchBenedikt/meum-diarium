# PWA Icons Generieren

## Automatische Generierung (empfohlen)

1. Öffne `generate-icons.html` im Browser
2. Die Icons werden automatisch generiert und als Preview angezeigt
3. Klicke auf "Alle Herunterladen" um die Icons zu speichern
4. Lege die heruntergeladenen Dateien (`icon-192x192.png` und `icon-512x512.png`) in diesem Verzeichnis ab

## Manuelle Generierung

Falls du eigene Icons verwenden möchtest:

### Anforderungen
- **icon-192x192.png**: 192x192 Pixel
- **icon-512x512.png**: 512x512 Pixel
- Format: PNG mit transparentem Hintergrund oder solidem Hintergrund
- Empfohlener Stil: Einfach, klar erkennbar, funktioniert auf verschiedenen Hintergründen

### Empfohlene Tools
- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- Adobe Illustrator / Figma / Sketch

### Tipps für gute PWA Icons
1. **Einfaches Design**: Icons sollten auch in kleinen Größen erkennbar sein
2. **Keine feinen Details**: Vermeiden Sie zu detaillierte Grafiken
3. **Kontrast**: Guter Kontrast zwischen Vorder- und Hintergrund
4. **Safe Zone**: Halte wichtige Elemente im inneren 80% Bereich (für maskable icons)
5. **Testing**: Teste die Icons auf verschiedenen Geräten (iOS, Android) und in Hell/Dunkel-Modi
